# Portfolio.Frse

Hi! This should in future be my portofolio made in Next.JS. It's currently in development but if you wanna try ... 

### Installation

The Project requires [Node.js](https://nodejs.org/en/) v10.13+ to run.

Install the dependencies.

```sh
$ cd my-portfolio
$ npm install 
```
```sh
$ cd my-portfolio
$ yarn install
```

Start the Server 
```sh
$ yarn dev
```

Start the Storybook 
```sh
$ yarn storybook
```
